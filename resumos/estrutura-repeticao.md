# 🧠 Resumo: Estrutura de Repetição (Loops / Laços)

Servem para executar o mesmo bloco de código várias vezes, evitando que você tenha que digitar a mesma linha repetidamente.

ENQUANTO (enquanto ... faca): Repete o bloco enquanto uma condição for verdadeira. É ideal quando você não sabe exatamente quantas vezes o loop vai rodar (ex: rodar o sistema até o usuário digitar "Sair").

PARA (para ... de ... ate ... faca): Repete o bloco por um número fixo e conhecido de vezes, usando uma variável contadora.

Snippet de código
// Exemplo de laço PARA (Contar de 1 até 5)
para contador de 1 ate 5 faca
   escreval("Número: ", contador)
fimpara
