# 🧠 Resumo: Estruturas Condicionais e de Repetição

Este resumo aborda como controlar o fluxo de um algoritmo, permitindo que o computador tome decisões e repita blocos de código de forma inteligente.

## 🔀 1. Estruturas Condicionais (Tomada de Decisão)
Servem para o programa decidir qual caminho seguir baseado em uma condição (Verdadeira ou Falsa).

* **Simples / Composta (`se ... entao ... senao`):** Avalia uma condição. Se for verdadeira, faz uma coisa; se for falsa, faz outra.
  * *Exemplo:* Verificar se o aluno foi aprovado.

```portugol
// Exemplo no VisuAlg
se (nota >= 7) entao
   escreval("Aprovado!")
senao
   escreval("Reprovado.")
fimse
